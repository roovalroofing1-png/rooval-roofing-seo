## Project Structure

```
schema/          LocalBusiness schema markup (added via WPCode)
meta/            Rank Math meta descriptions for all pages
snippets/        WPCode PHP snippets applied to the site
blog-posts/      SEO blog post content (published to WordPress)
```

## How to Revert a Change

Each file in `snippets/` corresponds to a WPCode snippet on the site.
To revert: go to WP Admin → Code Snippets → find the snippet → toggle it off or delete it.

To re-apply: paste the file content into a new WPCode snippet and activate it.
